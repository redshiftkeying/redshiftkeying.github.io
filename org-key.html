org模式备忘

快捷键的备忘录

tab的用法

使用tab和键可以折叠和隐藏对应的视图。 S-tab用来展开可收缩全局的条目，不包含内容。

标题间跳转

C-c C-n 下个标题 C-c C-p 上个标题 C-c C-f 下个同级的标题 C-c C-b 上个同级的标题 C-c C-u 回到上层标题

标题操作

M-RET 创建新的同级标题 M-S-RET 插入一个同级别的TODO标题 M-左／右 调整标题的级别 M-S-上／下 将当前标题及内容在树中移动 C-x n s／w 进入／弹出当前子目录

稀疏树

突出显示以正则方式匹配出来的信息以及其标题 C-c / 它会提示再输入一个字符来选择稀疏树的创建命令。 C-c / r 触发后，会提示输入匹配串，并且将所有匹配的项显示成稀疏树。所有的匹配项都会高亮显示；按 C-c C-c 取消高亮。

文本的操作

无序列

通过使用“-”、“+”和“*”来创建没顺序的文本内标题

有序列

通过使用“1.”、“1)“为开头创建顺序序列

描述序列

通过使用“::”来将内容包裹其中,如（link::描述）这样的组成

表格操作

首先要以“｜”开始(注意要用半角符号) 例如：

ID 名称 备注
1 测试 无
2 长名称 长备注测试测试。。。
3 更长的名称 无

看来处理中文的时候会走样。 C-c C-c 可以自动调节表格 TAB 移动到下一个单元格 S-TAB 移动到上一个单元格 RET 移动到下一行 M-左／右 左右移动当前列 M-上／下 上下移动当前行 M-S-上 删除当前行 M-S-下 插入新行 M-S-左 删除列 M-S-右 插入列

标签

相当与TODO功能中的@something 本标题带标签"test" C-c C-q 插入标签到标签列 C-c C-c 当光标在标题上时，同上功能 C-c / m 用匹配标签搜索的所有标题构造一个稀疏树。带前缀参数C-u时，忽略所有还是TODO行的标题。 C-c a m 用所有议程文件匹配的标签构造一个全局列表。 C-c a M 用所有议程文件匹配的标签构造一个全局列表，但只搜索 TODO 项，并强制搜索所有子项（见变量 org-tags-match-listsublevels）。

基本使用
