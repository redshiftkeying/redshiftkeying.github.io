设置

首先，你需要安装有 emacs (在版本 24 下测试，在 23 下可能也行) 和 org-mode (在版本 7.8.09 下测试，其它的 可能 也行)。然后你有两种方法可以安装 o-blog.

使用 el-get

要使用 el-get, 安装起来很容易。你所要做的不过是将 o-blog 添加到 el-get-sources 中，然后执行：

M-x el-get-install o-blog

如果你使用 el-get 4 或更高，你只需要执行:

M-x el-get-install o-blog

手动安装

首先从 Github 克隆 git 仓库源码。 

cd ~/.emacs.d
git clone https://github.com/renard/o-blog.git

将如下内容添加到你的 ~/.emacs.d/init.el 文件中: 

(add-to-list 'load-path "~/.emacs.d/o-blog")
(require 'o-blog)

发布博文

打开文件 ~/.emacs.d/o-blog/example/sample.org 然后执行 M-x org-publish-blog. 输出的站点默认保存在 ~/.emacs.d/o-blog/out.

详情参见 tips.
